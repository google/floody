Floody Security and Privacy Policy

Overview

The Floody tool is an HTML interface which allows users to connect to Campaign Manager via the Campaign Manager API. Users can only connect to Campaign Manager if they already have a User Profile set up in Campaign Manager as well as the permissions required to manage Floodlight activities.

Requirements

In connecting and managing Floodlight-related content in Campaign Manager via API, the Floody tool does not do anything that is not covered by Campaign Manager security and privacy policies. In fact, Floody accesses very few areas that are actually available via API.

Usage of permissions

Granted Google permissions are used as follows, your security tokens are not stored.

Google Profile is used only to show user photo and name on the app, it is not logged or stored.

Spreadsheets permission is used to ascertain your access to the given spreadsheet and allow you to share the spreadsheet with others.

DCM Trafficking permission is used to perform action on your behalf in Campaign Manager using the API.

Google Tag Manager permission is used to perform action on your behalf to push Floodlight Activities to GTM.
